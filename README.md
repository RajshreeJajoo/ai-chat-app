AI-Powered Career Mentor

A full-stack AI-powered chat application built with Next.js that provides real-time career guidance using AI.

 Features
💬 Real-time AI Chat: Streamed responses ke saath instant communication.
⚡ High Performance: Next.js App Router ke saath optimized aur fast loading.
🎨 Responsive Design: Tailwind CSS ke saath kisi bhi device par perfect view.
🧠 Smart AI Logic: Gemini API dwara powered, jo aapke Career Mentor app ko smart banata hai.
🎙️ Voice Chat Integration: Hands-free interaction ke liye integrated Speech-to-Text aur Text-to-Speech.
💾 Persistent History: MongoDB ke saath aapki chat history hamesha save rehti hai.
🌐 Cloud Ready: Vercel par seamless deployment.


🛠️ Tech Stack
Frontend: Next.js 14, React, TypeScript
Styling: Tailwind CSS
Backend: API Routes (Next.js)
Database: MongoDB
AI Integration: Gemini API
