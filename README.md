AI-Powered Career Mentor

A full-stack AI-powered chat application built with Next.js that provides real-time career guidance using AI.

🚀 Features
💬 Real-time AI chat with streaming responses
⚡ Fast and optimized with Next.js App Router
🎨 Responsive UI using Tailwind CSS
🧠 Smart AI responses using Gemini API
💾 Chat persistence using MongoDB
🌐 Deployed on Vercel


🛠️ Tech Stack
Frontend: Next.js 14, React, TypeScript
Styling: Tailwind CSS
Backend: API Routes (Next.js)
Database: MongoDB
AI Integration: Gemini API
